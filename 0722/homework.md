# 1. 이름 공간 (Namespace)

```
Python에서 변수를 찾을 때 접근하는 이름 공간을 순서대로 작성

1. Local scope : 함수
2. Enclosed scope : 특정 함수의 상위 함수
3. Global scope : 함수 밖의 변수, Import 모듈
4. Built-in scope : 파이썬 안에 내장되어 있는 함수 또는 속성
이를 LEGB Rule이라고 부른다.
```
